# bwb
The greatest card game of All Time. Based off of the original Buddies with Benefits card game, this project will be an Android port to the legendary card game made entirely with a custom-made game engine!

## Build Instructions

The project runs on Android Studio, install the latest version [here](https://developer.android.com/studio). Afterwards, clone the repository, open in Android Studio, and sync Gradle so the project files are properly built.
